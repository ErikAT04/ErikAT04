## Featured Projects
### Magik Antivirus
Antivirus developed as Final Project from the Multiplatform App Development Higher Studies, based in file analysis and putting files in and releasing them from quarantine.
- Technologies: FastAPI (Python) as BackEnd, Flutter (Dart) as FrontEnd and MySQL as database.
- Functionalities: File analysis, putting in and releasing files from quarantine, directory exclusion and both user and linked device management.
- Featured content: Dart multithread programming via Isolates, file cypher and comparing hashes from database.

FullStack date manager developed as main project the first year of the Multiplatform App Development Higher Studies.
